Чтобы использовать шаблон интрасети, необходимо включить проверку подлинности 
Windows и отключить анонимную проверку подлинности.

Подробные инструкции (в том числе инструкции для IIS 6.0) см. на странице
http://go.microsoft.com/fwlink/?LinkID=213745

IIS 7
1. Откройте диспетчер служб IIS и перейдите на свой веб-сайт.
2. В представлении "Просмотр возможностей" дважды щелкните пункт "Проверка 
   подлинности".
3. На странице "Проверка подлинности" выберите "Проверка подлинности Windows". 
   Если вариант "Проверка подлинности Windows" отсутствует, необходимо 
   убедиться, что решение "Проверка подлинности Windows" установлено на сервере.
        Включение проверки подлинности Windows:
 а) В панели управления выберите "Программы и компоненты".
 б) Щелкните "Включение и отключение компонентов Windows".
 в) Перейдите последовательно Службы IIS | Службы Интернета | Безопасность
    и убедитесь, что выбран узел проверки подлинности Windows.
4. На панели "Действия" щелкните пункт "Включить", чтобы включить проверку 
   подлинности Windows.
5. На странице "Проверка подлинности" выберите "Анонимная проверка подлинности".
6. На панели "Действия" щелкните пункт "Отключить", чтобы отключить анонимную 
   проверку подлинности.

IIS Express
1. Щелкните правой кнопкой мыши проект в Visual Studio и выберите команду 
   "Использовать IIS Express".
2. В обозревателе решений щелкните свой проект, чтобы выбрать его.
3. Если панель "Свойства" не открыта, откройте ее (F4).
4. В области "Свойства" для своего проекта:
 а) Для варианта "Анонимная проверка подлинности" задайте значение "Отключена".
 б) Для варианта "Проверка подлинности Windows" задайте значение "Включена".

Можно установить IIS Express с помощью установщика веб-платформы Майкрософт:
    Для Visual Studio: http://go.microsoft.com/fwlink/?LinkID=214802
    Для Visual Web Developer: http://go.microsoft.com/fwlink/?LinkID=214800
